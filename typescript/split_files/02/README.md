# typescript / split_files / 02

## cli

* compile 

> with typescript compiler

```bash
tsc src/main.ts --target es5 --module "CommonJS" --outDir dist
```

* run

```bash
node dist/main
```

* clean up

> *nix only

```bash
rm -rf dist
```


## npm

* compile

* run compiled

* clean up
