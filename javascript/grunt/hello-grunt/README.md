# Commands on Mac

## Create commands list

```
mkdir hello-grunt

cd hello-grunt

touch README.md

npm init

touch index.js 

touch Gruntfile.js

npm install grunt --save-dev

npm install grunt-contrib-jshint --save-dev

npm install grunt-contrib-uglify --save-dev

npm install grunt-contrib-clean --save-dev
```

## project cloned from git

```
npm install
```

## run command

```
grunt
```
