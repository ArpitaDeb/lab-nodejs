# Commands on Mac

## Create commands list

```
mkdir hello-watch-grunt

cd hello-watch-grunt

touch README.md

npm init

touch Gruntfile.js

npm install \
  grunt \
  grunt-contrib-jshint \
  grunt-contrib-watch \
  --save-dev
```

## project cloned from git

```
npm install
```

## run command

`grunt` - to check jshint.

or 

`grunt w` - to watche files change.
